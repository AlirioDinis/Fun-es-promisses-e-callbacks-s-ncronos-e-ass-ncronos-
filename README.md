Neste único projecto eu decidi mostrar a manipulação de 3 funções no estudo de códigos assíncronos que eu tenho estado a estudar na plataforma #The odin project na trilha de Fullstack Js
Começando pela função promisse que funciona junto de um metódo chamado .then() que registra um callback quando a promesa for resolvida.
E em seguida temos dois callbacks um deles é síncrono e o outro assíncrono:
o nosso callback síncrono ele exibe um resultado de uma operação digitada pelo usuário em que essa operação é está junta de uma função .eval() que se a string for uma expressão, ele calcula o resultado;
e o nosso callback assincrono que executa uma tarefa após o termino de uma função assíncrona 'SetTimeout'

Essas 3 funções escritas em js estão sendo exportadas para o nosso arquivo js principal na qual estão sendo dentro de um 'if' para opção de escolha do usuário!

Dica de ouro: executem em um navegador com a aba inspencionar aberta na janela console.
